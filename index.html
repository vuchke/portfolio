About

Myself

Hello I'm Goran, a front-end developer based in Serbia. I enjoy creating things that live on the internet, whether that be websites, applications, or anything in between. My goal is to always build products that provide pixel-perfect, performant experiences. I started learning front-end development by myself in the February of 2019, it started like a hobby but became a passion. I'm able to work with the following technologies:

  • CSS 3, Animations, Flexbox, Responsive Web Design
  • JavaScript, ECMAScript 6
  • Vue, Vue Router, Vuetify
  • Bootstrap
  • Wordpress

  • Photoshop
